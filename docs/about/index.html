About the Author or Blog

I'm James a full stack web developer from Blackwood in South Wales, I'm 34 years of age, married back in 2018 and have two daughters due in July 2021. My tech stack is PHP, MySql, Javascript, HTML and CSS and Docker. I work mainly with Laravel a PHP framework and Vue.js which is a Javascript framework.

I've been a full stack developer since 2018, and I started learning PHP back in December 2017, learning from TeamTree House. Within 6 months of being self taught, I then secured myself a full time job as a PHP developer.

In my spare time, I enjoy learning to code and building things in my spare time, please do checkout my Github

My aim for the programming blog is to write about topics that I look at or am currently learning, so things I find out from trial and error, I can feedback through this blog, so others can read and learn from it too.
