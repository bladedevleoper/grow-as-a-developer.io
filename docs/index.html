What is Golang?

Hello Golang, nice to meet you, I'm James

So I have been meaning to write a blog for sometime, and I have been programming with PHP, Javascript coming up to 2 years. So I've been looking at learning a new language, and always wondered what it would be like to learn Golang.

What Is Golang?

Go or as others would say Golang, is a an open source programming language that allows to build reliable and efficient software. Go was created by a team of developers at Google. They are Robert Griesemer, Rob Pike and Ken Thompson. Golang is a staticically and a strongly typed language

What can Golang be used for?

Golang can be used in many different ways:

  • Build Web Applications
  • CLI Applications
  • Automate tasks on the server

Like Python and PHP, Go too is a general purpose language which means it can be used in a wide variety of development tasks.

Where Can I Learn Golang?

There are a number of resources to start to learn Go:

Youtube

I will always search Youtube, but who doesn't right? I found the freecodecamp.org youtube channel to be a really good starting point. I had found a 7 hour Golang video created by Michael Van Sickle. For a Go beginner, I found this very informative, starting from the basic syntax and how to set your local environment up.

Please note, as of this videos creation the version of Go is at 1.14 , there is a newer version of Go which is 1.16 to download follow and pick your OS

Since Go 1.16 is released (February 2021), we can start to use go.mod, which allow is Go's dependency management. An following a project setup in the video, we do not have to use all the directories. I would advise to look at the following article on golang.org on how to organise your code

Go Documentation

The Golang.org is another nice place to get started, where it takes you on a step by step through the Go language. If you want to pick up some extra reading on a particular section.

How can I practice Go?

Go Playground

The Go Playground has an in browser terminal which allows you to play with Go by writting the code and pressing run. It temporarely compiles and runs your code in the browser for instant feedback.
